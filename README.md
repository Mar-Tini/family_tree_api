# Family Tree API

## Description

`family_tree_api` est une API qui permet de gérer un **arbre généalogique**.  
Elle fournit des endpoints pour créer, modifier, supprimer et récupérer des membres et leurs relations dans une famille.  

Idéal pour des applications frontend comme React  qui affichent un arbre généalogique interactif.

---

## Fonctionnalités
- Ajouter un membre de la famille
- Modifier les informations d’un membre
- Supprimer un membre
- Définir des relations (parent, enfant, conjoint)
- Récupérer l’arbre généalogique complet ou un sous-arbre
- Rechercher un membre par nom
